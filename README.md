# Decision Trees and Random Forests - Task 5

## Objective

Learn and implement tree-based machine learning models for classification using:

* Decision Tree Classifier
* Random Forest Classifier

## Tools & Libraries

* Python
* Scikit-learn
* Pandas
* NumPy
* Matplotlib
* Seaborn

## Dataset

This notebook uses the Breast Cancer dataset available in Scikit-learn.

## Tasks Performed

1. Loaded and explored the dataset
2. Split the dataset into training and testing sets
3. Trained a Decision Tree Classifier
4. Visualized the Decision Tree
5. Controlled overfitting using max depth
6. Trained a Random Forest Classifier
7. Compared model accuracy
8. Analyzed feature importance
9. Evaluated models using cross-validation

## Results

* Decision Tree provides interpretable classification results.
* Random Forest improves accuracy and reduces overfitting.
* Feature importance helps identify the most impactful features.

## How to Run

1. Open the notebook in Google Colab or Jupyter Notebook.
2. Install dependencies if required:

```bash
pip install scikit-learn pandas matplotlib seaborn
```

3. Run all cells step by step.
